# SCN-LIB-001

## 主題

小型圖書櫃管理

---

## 🧩 背景設定（30 秒開場）

公司團隊有一個共享書櫃，大家會互相借書。

但實際情況是：

- 書被拿走不知道誰借的

- 想找書時不確定還在不在

- 有些書借很久沒人記得

因此做一個**最小可用管理系統**。

---

## 第一段：建立書籍

> 先把書櫃裡的書建進系統。

操作：

1. 點擊「新增書籍」

2. 輸入書名與作者

3. 設定庫存數量

新增幾本書，例如：

- Clean Code

- Domain-Driven Design

- Designing Data-Intensive Applications

畫面出現書籍清單。

---

## 第二段：借書

> Samson 想借 Clean Code。

操作：

1. 在書籍清單找到 Clean Code

2. 點擊「借出」

3. 輸入借閱人姓名

畫面更新：

- 可借數量減少

- 狀態改為「已借出」

---

## 第三段：還書

> Samson 把書還回來了。

操作：

1. 點擊該筆借閱紀錄的「歸還」

2. 書籍狀態更新

3. 可借數量恢復
